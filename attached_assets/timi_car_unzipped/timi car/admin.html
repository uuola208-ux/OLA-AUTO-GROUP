📸 How to add images:
Option 1: Image Links
1. Get image URL (from Google Images, Unsplash, etc)
2. Right-click → "Open image in new tab" to get DIRECT link
3. Paste the URL (should start with http)

Option 2: Upload from PC
1. Click "PC UPLOAD" tab
2. Select image files from your computer
3. Multiple files supported
Note: Files are converted to embedded images
